# Inloggen in het CMS

![Inloggen](/img/basics/inloggen.png)

1. Ga naar de inlogpagina via de aangeleverde URL
2. Vul je e-mailadres en wachtwoord in, en klik op "Inloggen"

:::info

Wachtwoord vergeten? Klik op "Wachtwoord vergeten?", vul vervolgens je gebruikersnaam in. Je ontvang per e-mail instructies om je wachtwoord opnieuw in te stellen.

:::
