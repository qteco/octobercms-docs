# CMS instellingen

![CMS instellingen](/img/basics/cms-instellingen.png)

Je kunt binnen het CMS altijd naar de CMS instellingen gaan door rechtsboven op je profielfoto te klikken en vervolgens op "CMS voorkeuren" te klikken:

![Ga naar CMS voorkeuren](/img/basics/cms-instellingen-link.png)

Op deze pagina kun je je weergavetaal en tijdzone wijzigen.

Het is aanbevolen om in het "Tijdzone" veld je huidige tijdzone in te vullen, dus bijv. "Europe/Amsterdam".

Het is niet aanbevolen om instellingen in het "Code editor"-tabblad te wijzigen.
