# Het October CMS dashboard

![Dashboard](/img/basics/dashboard.png)

Zodra je succesvol bent ingelogd kom je altijd op het dashboard terecht. Op het dashboard kun je in één oogopslag een aantal belangrijke details over de website inzien.

**Systeem build:** Dit is de geïnstalleerde versie van October CMS en tevens een snelkoppeling naar de "Beheer updates" pagina.

**Gebeurtenis logboek:** Dit is de systeemlog, hier worden o.a. foutmeldingen weergegeven.

**Voerzoek logboek:** Hier worden alle “bad requests” geregistreerd, bijvoorbeeld als een bezoeker naar een pagina navigeert die niet (meer) bestaat.

**Online sinds:** De datum waarop October CMS is geïnstalleerd.
