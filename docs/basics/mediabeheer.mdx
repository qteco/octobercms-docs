# Mediabeheer

![Mediabeheer](/img/basics/mediabeheer.png)

Op deze pagina kun je alle mediabestanden van de website beheren.

## Bovenste toolbar

![Bovenste toolbar](/img/basics/mediabeheer-bovenste-toolbar.png)

Met de bovenste toolbar kun je:

-   mediabestanden uploaden, verplaatsen en verwijderen;
-   de mediabibliotheek verversen om nieuwe uploads te tonen;
-   mappen aanmaken;
-   tussen lijst- en 2 soorten rasterweergave wisselen;
-   mediabestanden zoeken.

### Mediabestanden uploaden

Om een of meerdere mediabestanden te uploaden klik je in de bovenste toolbar op "Uploaden", vervolgens verschijnt er een dialoogvenster waar je een of meerdere bestanden kunt kiezen.

Je kunt o.a. de volgende soorten bestanden uploaden:

-   afbeeldingen (`.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`, `.ico`, `.svg`);
-   video’s (`.mp4`, `.m4v`, `.mov`);
-   audiobestanden (`.mp3`, `.wav`);
-   documenten (`.pdf`, `.docx`, `.xlsx`, `.csv`, `.txt`).

:::note

Gebruik geen spaties in de namen van je mediabestanden, dit kan problemen veroorzaken. Het is aanbevolen om je mediabestanden [URL-vriendelijke](https://en.wikipedia.org/wiki/Letter_case#Kebab_case) namen te geven.

:::

### Mappen toevoegen

Je kunt in het mediabeheer mappen aanmaken om je mediabestanden te organiseren.

Om een nieuwe map aan te maken klik je in de bovenste toolbar op "Map toevoegen", er verschijnt vervolgens een venster waar je de naam van de map kunt invoeren.

:::note

Gebruik geen spaties in de namen van je mappen, dit kan problemen veroorzaken. Het is aanbevolen om je mappen [URL-vriendelijke](https://en.wikipedia.org/wiki/Letter_case#Kebab_case) namen te geven.

:::

:::caution

Wanneer je een mediabestand naar een map verplaatst verandert de URL van het mediabestand, het is dus niet aanbevolen om dit te doen nadat de het mediabestand al op de website wordt getoond.

Wanneer je een mediabestand wat al op de website wordt getoond in een map plaatst, zul je dit mediabestand op de betreffende pagina(‘s) opnieuw moeten plaatsen.

:::

### Mediabestanden / mappen verplaatsen

Je kunt mediabestanden en mappen verplaatsen naar een andere map door het betreffende bestand of map te selecteren, en vervolgens op "Verplaatsen" te klikken. Er opent vervolgens een venster waar je de gewenste bestemming kunt selecteren.

:::caution

Wanneer je een mediabestand in een map plaatst verandert de URL van het mediabestand, het is dus niet aanbevolen om dit te doen nadat de het mediabestand al op de website wordt getoond.

:::

### Mediabestanden verwijderen

Om een mediabestand te verwijderen selecteer je het betreffende mediabestand en klik je in de bovenste toolbar op "Verwijderen".

Mediabestanden die je via het mediabeheer verwijdert zijn permanent verwijderd, wees dus voorzichtig.

## Linker zijbalk

![Linker zijbalk](/img/basics/mediabeheer-linker-zijbalk.png)

De linker zijbalk is je filter, je kunt hier:

-   op een bepaald type mediabestand filteren;
-   sorteren op titel, grootte en laatst gewijzigd;
-   de volgorde wijzigen op oplopend en aflopend.

## Mediabibliotheek

![Mediabibliotheek](/img/basics/mediabeheer-mediabibliotheek.png)

Dit is de lijst van alle geüploade mediabestanden op de website.

Zodra je op een mediabestand klikt, komt er aan de rechterkant een zijbalk tevoorschijn met een voorvertoning, en overige informatie over het mediabestand.

Zodra je hier een afbeelding uploadt wordt er automatisch een “.webp”-variant gegenereerd. Dit is een afbeeldingsformaat wat gemiddeld 39% kleiner is zonder kwaliteitsverlies, dit resulteert in een webpagina die sneller laadt. Deze .webp-variant wordt automatisch getoond wanneer de browser van de bezoeker het .webp formaat ondersteunt, je hoeft deze .webp afbeeldingen dus niet handmatig aan de webpagina toe te voegen. Het is aanbevolen om zowel de normale als de .webp variant te verwijderen wanneer je een afbeelding verwijdert.

### Naam van een mediabestand / map wijzigen

Als je met je cursor over een mediabestand / map zweeft, komt er aan de rechterkant een “>\_” icoontje tevoorschijn. Je kunt hierop klikken om de naam van een mediabestand / map te wijzigen. Let op dat je bij mediabestanden niet de bestandsextensie (bijv. .jpg) wijzigt.

:::caution

Wanneer je een mediabestand hernoemt verandert de URL van het mediabestand, het is dus niet aanbevolen om dit te doen nadat de het mediabestand al op de website wordt getoond.

Wanneer je een mediabestand wat al op de website wordt getoond hernoemt, zul je dit mediabestand op de betreffende pagina(‘s) opnieuw moeten plaatsen.

:::
