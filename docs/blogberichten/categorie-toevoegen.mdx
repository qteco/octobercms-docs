# Nieuwe categorie toevoegen

![Nieuwe categorie toevoegen](/img/blogberichten/categorie-toevoegen.png)

## Invoervelden

**Naam:**  
De naam van de categorie.

**Slug:**  
De URL van de categorie.  
De slug wordt automatisch gegenereerd op basis van de naam van de categorie, maar je kunt ook een andere slug invoeren.

**Description**  
Omschrijving van de categorie, dit veld is alleen voor inten gebruik.
