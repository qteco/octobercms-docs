# Blogberichten beheren

![Blogberichten beheren](/img/blogberichten/blogberichten-beheren.png)

Op deze pagina kun je alle blogberichten van de website beheren. Klik op het gewenste blogbericht om deze te bewerken / verwijderen, of klik op "Nieuw artikel" om een nieuw blogbericht toe te voegen.
