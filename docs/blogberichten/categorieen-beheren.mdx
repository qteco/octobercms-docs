# Categorieën beheren

![Categorieën beheren](/img/blogberichten/categorieen-beheren.png)

Op deze pagina kun je alle categorieën voor de blogberichten beheren. Klik op de gewenste categorie om deze te bewerken / verwijderen, of klik op "Nieuwe categorie" om een nieuwe categorie toe te voegen.
