Menu bewerken

Het is niet aanbevolen om de velden "Naam" en "Code" te wijzigen, omdat deze waarden nodig zijn om het menu op de website weer te geven.

Beschikbare acties wanneer je met je cursor over een menu item zweeft:

Menu item toevoegenâ€‹

Er zijn 2 manieren om een menu item toe te voegen. Door onderaan op "Item toevoegen" te klikken, of door met je cursor over een bestaand menu item te zweven en vervolgens op "Subitem toevoegen" te klikken.

  • Bij optie 1 wordt het menu item in het hoogste niveau aan het menu toegevoegd
  • Bij optie 2 wordt het menu item als sub-item aan een bestaand menu item toegevoegd

Invoerveldenâ€‹

Titel
De titel van het menu-item, wordt op de website weergegeven

Type
Het type menu-item:

  • URL: Een reguliere URL, bijvoorbeeld een verwijzing naar een andere website.
  • Header: Menu item zonder bestemming
  • CMS pagina: Verwijzing naar een maatwerk pagina
  • Statische pagina: Verwijzing naar een contentpagina
  • Alle statische pagina's: Verwijzing naar alle contentpagina's

Afhankelijk van het geselecteerde type zijn er een aantal extra invoervelden:

"URL" geselecteerd:

URL
De URL van de bestemming

"CMS pagina" geselecteerd:

Referentie
De gewenste CMS pagina

"Statische pagina" geselecteerd:

Referentie
De gewenste contentpagina

Accepteer geneste items
Wanneer de gewenste contentpagina sub-pagina's heeft kun je kiezen door deze items ook weer te geven door deze optie aan te vinken.

Vervang dit item door de gegenereerde subitems
Indien bovenstaande optie is ingeschakeld kun je deze optie inschakelen door het hoofd-item te verbergen en alleen de sub-items te tonen.

"Alle statische pagina's" geselecteerd:

Vervang dit item door de gegenereerde subitems
Je kunt deze optie inschakelen om het hoofd-item te verbergen en alleen de sub-items te tonen.

Wanneer deze optie is ingeschakeld wordt het menu-item niet in het menu weergegeven.

Attributes â†’ Code
Geeft het menu-item een unieke code. Niet aanpassen.

Attributes â†’ CSS Class
Voegt een CSS klasse aan het menu-item toe. Niet aanpassen.

Attributes â†’ Exernal link
Wanneer deze optie is ingeschakeld wordt het de link op een nieuwe pagina geopend wanneer iemand er op klikt.

Menu item verplaatsenâ€‹

Om een menu item te verplaatsen kun je met je cursor over een bestaand menu item zweven, het icoontje met de 3 horizontale streepjes aan te klikken en vervolgens het menu item naar de gewenste locatie te slepen.

Menu item verwijderenâ€‹

Om een menu item te verwijderen kun je met je cursor over een bestaand menu item zweven en vervolgens op het prullenbak-icoontje te klikken.
